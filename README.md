# Mini Social Network

## Overview
This project is a mini social network application that allows users to create profiles, make posts, and interact with each other. It aims to provide a simple platform for social interactions, focusing on user privacy and security.

## Setup Instructions
1. **Clone the Repository**  
   Run the following command to clone the repository:  
   ```bash
   git clone https://github.com/Yorlinjjj/mini-social-network.git
   ```  

2. **Navigate to the Project Directory**  
   ```bash
   cd mini-social-network
   ```  

3. **Install Dependencies**  
   If you are using Node.js, run:  
   ```bash
   npm install
   ```  
   If you are using Python, make sure to set up a virtual environment and run:  
   ```bash
   pip install -r requirements.txt
   ```  

4. **Run the Application**  
   To start the application, run:  
   ```bash
   npm start  # for Node.js
   ```  
   or  
   ```bash
   python app.py  # for Python
   ```  

5. **Access the App**  
   Open your browser and go to `http://localhost:3000` or the specified port to access the application.
